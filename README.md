# book store

> it helps to add and read books online. you can comment, edit and remove the book. You can also track its progress

## Built With

- Reactjs



## Getting Started

**clone the project by running:

```
$ git clone https://github.com/JoffreyNK/book-store.git

cd book-store

npm install

npm start
```
